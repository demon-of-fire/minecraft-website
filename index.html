About This Wiki

Complete Minecraft Reference

Comprehensive data for all Minecraft versions from Classic to 1.21.1+:

  • Crafting Recipes - Every recipe with visual grid layouts
  • All Items - Complete item database with properties and rarity
  • Mob Drops - Exact drop rates and conditions for every mob
  • Commands - Full command reference with syntax and examples
  • Interactive Tools - Calculators, enchantment checker, brewing guide, and more
  • References - Enchantments, potions, biomes, and advancements
